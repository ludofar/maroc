Vocabulaire

  • Lundi : L-tnin
  • Mardi : T-tlat
  • Mercredi : L-arb3a
  • Jeudi : L-khmis
  • Vendredi : J-jm3a
  • Samedi : S-sbt
  • Dimanche : L-hdd

Phrases

  • L-yum l-tnin. (Aujourd’hui, c’est lundi.)
  • Ghdda t-tlat. (Demain, c’est mardi.)

Notes

Utilisez l-yum pour "aujourd’hui" et ghdda pour "demain".
