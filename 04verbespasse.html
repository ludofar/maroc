Règles de conjugaison

Au passé, les verbes réguliers prennent des suffixes selon le pronom :

  • Je : racine + -t
  • Tu (m.) : racine + -ti
  • Tu (f.) : racine + -ti
  • Il : racine (sans suffixe)
  • Elle : racine + -t
  • Nous : racine + -na
  • Vous : racine + -tu
  • Ils/Elles : racine + -u

Exemple : "Shrb" (Boire)

Pronom Conjugaison
Ana Shrbt
Nta/Nti Shrbti
Huwa Shrb
Hiya Shrbat
7na Shrbna
Ntuma Shrbtu
Huma Shrbu
