Règles de conjugaison

Pour conjuguer un verbe régulier au présent, utilisez ces préfixes et suffixes :

  • Je : N- + racine
  • Tu (m./f.) : T- + racine
  • Il : Y- + racine
  • Elle : T- + racine
  • Nous : N- + racine + -u
  • Vous : T- + racine + -u
  • Ils/Elles : Y- + racine + -u

Exemple : "Kla" (Manger)

Exercices

Conjuguez au présent :

  1. Shrb (Boire) - Je : ____
  2. Qra (Lire) - Ils : ____

Réponses : 1. Nshreb, 2. Yaqraw
