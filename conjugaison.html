Introduction

La conjugaison en darija est une étape clé pour parler et comprendre le marocain couramment. Cette page vous présente les conjugaisons des 20 verbes principaux aux temps présent, passé et futur, ainsi que les verbes irréguliers et un cours pour apprendre à conjuguer tout verbe à partir de sa forme infinitive.

Cours : Comment conjuguer un verbe en darija

Pour conjuguer un verbe en darija à partir de sa forme infinitive (indicatif), suivez ces étapes :

1. Identifier la racine du verbe

La plupart des verbes ont une racine de trois consonnes (ex. : K-T-B pour "écrire", K-L pour "manger"). Parfois, des voyelles sont ajoutées dans la forme infinitive (ex. : "Kteb", "Kla").

2. Conjugaison au présent

Ajoutez les préfixes et suffixes selon le pronom :

  • Je (Ana) : N- + racine
  • Tu (Nta/Nti) : T- + racine
  • Il (Huwa) : Y- + racine
  • Elle (Hiya) : T- + racine
  • Nous (7na) : N- + racine + -u
  • Vous (Ntuma) : T- + racine + -u
  • Ils/Elles (Huma) : Y- + racine + -u

Exemple avec "Kteb" : Nkteb, Tkteb, Ykteb, etc.

3. Conjugaison au passé

Modifiez la racine avec des suffixes :

  • Je (Ana) : racine + -t
  • Tu (Nta) : racine + -ti
  • Il (Huwa) : racine seule
  • Elle (Hiya) : racine + -at
  • Nous (7na) : racine + -na
  • Vous (Ntuma) : racine + -tu
  • Ils/Elles (Huma) : racine + -u

Exemple avec "Kteb" : Ktebt, Ktebti, Kteb, etc.

4. Conjugaison au futur

Utilisez "Ghadi" + la forme du présent :

  • Exemple avec "Kteb" : Ghadi nkteb, Ghadi tkteb, etc.

5. Exceptions

Les verbes irréguliers comme "Ja" (venir), "Msha" (aller) ou "Kan" (être) ont des formes spécifiques qui ne suivent pas ces règles. Reportez-vous aux tableaux ci-dessus pour les apprendre.

Exemple pratique : Si vous avez le verbe "Shuf" (voir) :

  • Présent : Nshuf, Tshuf, Yshuf...
  • Passé : Shft, Shfti, Shaf...
  • Futur : Ghadi nshuf, Ghadi tshuf...
